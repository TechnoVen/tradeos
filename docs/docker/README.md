```bash

chmod +x docker-run.sh
./docker-run.sh
```

```powershell
curl.exe -O https://raw.githubusercontent.com/marketcalls/tradeos/main/install/docker-run.bat
docker-run.bat
```

### Server Installation (Ubuntu/Debian)
```bash

chmod +x install-docker.sh
sudo ./install-docker.sh
```

## üê≥ Docker Configuration

### Current Setup

TradeOS uses a **multi-stage build** with the following configuration:

```yaml
services:
  tradeos:
    image: tradeos:latest
    container_name: tradeos-web

    ports:
      - "5000:5000"   # Web UI
      - "8765:8765"   # WebSocket

    volumes:
      - tradeos_db:/app/db
      - tradeos_log:/app/log
      - tradeos_strategies:/app/strategies
      - tradeos_keys:/app/keys
      - tradeos_tmp:/app/tmp
      - ./.env:/app/.env:ro

    shm_size: '2gb'  # For scipy/numba operations

    restart: unless-stopped
```

### Runtime Dependencies

The Docker image includes:
- **Python 3.12** (required)
- **libopenblas0** - BLAS/LAPACK for linear algebra
- **libgomp1** - OpenMP for parallel operations
- **libgfortran5** - Fortran runtime for scipy
- **numba 0.63.1** - JIT compilation
- **llvmlite 0.46.0b1** - LLVM bindings
- **scipy 1.17.0** - Scientific computing

## üß™ Testing

### Verify Installation
```bash
# Check container status
docker ps --filter "name=tradeos"

# Test numba/scipy
docker exec tradeos python -c "import numba, scipy; print('‚úì OK')"

# View logs
docker-compose logs -f
```

### Verify Shared Memory
```bash
docker inspect tradeos --format='{{.HostConfig.ShmSize}}'
# Should show: 2147483648 (2GB)
```

### Verify Volumes
```bash
docker inspect tradeos --format='{{range .Mounts}}{{.Destination}} {{end}}'
# Should include: /app/tmp /app/keys
```
